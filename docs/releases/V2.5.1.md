# DCSHUB V2.5.1 更新说明

<!-- dcshub-push-update -->

V2.5.1 是 DCSHUB 的一次大型架构与体验升级。本版本正式上线 VR 内置手册浮窗和本地语音输入，完成软件核心框架与超级手册 RAG 库重构，并作为面向所有已开启更新检查用户的建议更新版本发布。

## 本次大版本重点

- 原生 OpenXR VR 内置手册浮窗正式上线，可直接在 DCS VR 中查询操作步骤、查看引用原页。
- 新增 SenseVoice 本地语音输入，支持长按说话、松开识别和自动提问。
- 软件框架重构为 React/Electron、独立 .NET 10 Core Host 与 Windows/OpenXR 原生组件分层架构。
- 超级手册 RAG 库重构，强化机型隔离、章节边界、术语理解、来源优先级和证据审校。
- 按当前基准统计，本地手册平均搜索时间缩短约 60%，联网搜索时间缩短约 40%。
- 新增阿里云百炼千问和硅基流动 API 支持，并保留 DeepSeek。

## VR 内置手册浮窗

V2.5.1 将超级手册真正带入 DCS 的 OpenXR VR 环境：

- 在仪表板选择 VR 启动模式后，DCSHUB 会为本次 DCS 会话启用原生 OpenXR 手册层。
- 呼出浮窗时以当前视线方向建立位置；显示后固定在世界空间，不会一直黏着头部移动。
- 可拖动标题栏，让浮窗围绕用户做自然的弧形移动，并持续朝向用户。
- 桌面和 VR 分别记忆用户调整的窗口尺寸，切换模式或重启后恢复各自大小。
- VR 与桌面共用同一套本地问答、联网搜索、引用编号、原手册页面和图片放大交互。
- 右侧引用页面会随答案阅读位置保持可见，可使用滚轮切换引用页并点击放大。
- 重新呼出时会按新的视线方向定位，无需额外设置回中快捷键。

> 本版本更新了 OpenXR 原生组件。升级完成后请彻底退出并重新启动 DCS，再测试 VR 浮窗。

## 语音输入：使用方法

首次使用前：

1. 打开“设置 → 超级手册 → 内置手册窗口”。
2. 下载 SenseVoice 本地语音模型。
3. 选择要使用的麦克风。
4. 按需要设置手册呼出键，也可以绑定 Windows 游戏控制器按钮。新默认键为 `Ctrl+Alt+M`。

在 DCS 中使用：

1. **短按呼出键**：显示或隐藏内置手册浮窗。
2. **长按呼出键并说话**：浮窗会显示正在录音。
3. **松开按键**：系统补录很短的尾音后开始本地识别。
4. 识别文本会先进入输入框，并保留 **1.6 秒修改时间**。
5. 需要立即发送时可直接按 `Enter`；开始修改文字会取消倒计时；不操作则自动提交本地手册问答。
6. 需要联网资料时，可修改识别文本后按 `Ctrl+Enter`，或点击联网搜索按钮。

语音后处理现已与超级手册术语库联动：

- `iPhone / F fourteen / F十四` 会归一为 `F-14`。
- 支持 F/A-18C、F-16C、F-15、A-10、AH-64D、JF-17、Su/MiG 系列及常见直升机、运输机和二战机型的中英文拆读。
- “节达姆、杰达姆”等常见识别结果会归一为 `JDAM`。
- DCS 武器、传感器、航电和英文缩写会在检索前统一为标准术语。

## 软件框架重构

- Electron 主进程收敛为窗口、IPC、热键和服务编排层。
- 进程监控、DCS 状态和语音录制下沉到独立 .NET 10 Core Host。
- OpenXR 浮窗由独立 C++ API Layer、帧桥和 D3D11 纹理链路提供。
- 模块生命周期、日志、异常诊断和关闭清理统一管理。
- 修复主窗口关闭后隐藏浮窗导致 Electron、Core Host 或 VR Bridge 残留的问题。
- 桌面启动不会启用 VR OpenXR 层；VR 组件只在对应模式和生命周期内注册。
- 旧默认 `F9` 手册热键自动迁移为 `Ctrl+Alt+M`，关闭内置手册功能时会停止键盘 Hook，降低与 TrackIR 的冲突。

## 超级手册 RAG 库重构

- 文档入库、解析、章节书签、页面切块、搜索索引和预览缓存重新分层。
- 先按机型锁定手册，再按武器、系统、任务和真实 PDF 章节子树检索，避免跨机型或跨场景拼接。
- 按 Chuck's Guides、DCS/开发商官方手册、用户资料和非全拟真官方资料执行来源优先级。
- 长流程按准备、设置、执行、反馈和限制检查覆盖度，减少只回答局部页面的问题。
- 武器家族、具体型号、导引方式、乘员席位和工作模式保持独立边界。
- 答案经过证据审校，每个事实与引用来源绑定；无来源内容不会进入最终答案。
- 本地答案、联网答案、索引和 PDF 页面预览使用独立持久缓存。
- 对无意义或缺少可核实主题的问题不再用弱相关页面强行回答，而是提示用户使用联网搜索。

经过检索编译、候选压缩、章节边界和缓存策略优化，按当前基准统计：

- 本地手册平均搜索时间缩短约 **60%**。
- 联网搜索平均完成时间缩短约 **40%**。

## AI API 支持

- **DeepSeek**：支持本地手册问答与原生联网研究。
- **阿里云百炼千问**：支持本地手册问答、模型列表读取和原生联网研究。
- **硅基流动**：支持兼容 OpenAI 接口的本地手册模型和模型列表读取；当前不提供原生联网搜索入口。
- 本地手册与联网搜索可以分别选择供应商、模型和思考强度。
- API Key 继续使用 Windows 当前用户凭据加密保存。

## 其他重要修复

- 修复长按语音热键被误判为短按而关闭浮窗。
- 松手后增加录音尾音，降低句尾被截断的概率。
- 修复 F-14 被识别为 iPhone、JDAM 被识别为“节达姆”等术语错误。
- 修复弱相关检索可能把武器问题回答成 ATFLIR 等无关章节。
- 修复右侧引用图片不能随阅读位置实时保持可见。
- 修复旧答案中的独立“成功判断”被错误转换成额外编号步骤。
- 修复窗口关闭、OpenXR 注册和后台子进程清理的多项生命周期问题。

## 升级说明

- 可以从 V2.2.1 及更早公开版本直接安装升级到 V2.5.1。
- 用户设置、软件预设、模组预设、手册库和缓存目录不会被主动清除。
- 更新提示只负责通知并打开官方 GitHub Release 页面，不会在后台静默下载安装。
- 正式公开版本以 GitHub Release 中的 Windows 安装程序为准。
- 当前安装程序尚未进行商业代码签名，Windows 可能显示未知发布者提示。
