# DCSHUB V2.1.23 更新说明

V2.1.23 汇总了远端 V2.1.6 之后的本地开发成果，重点完成超级手册检索可靠性、桌面与 VR 内置窗口、软件管理、安装发布和界面品牌的系统性升级。本次合并只更新源码和仓库文档，不创建 GitHub Release，也不会触发现有用户的主动更新提示。

## 超级手册检索与回答

- 重构检索流水线，按机型、任务、资料来源和页面连续性分阶段召回。
- 机型用于严格限定候选文档，不再作为正文高分词，避免 F-16、F/A-18C、AH-64D 等机型互相串文档。
- 增加 DCS 和军事术语归一化、中文口语问题扩展及常见缩写理解。
- 多份资料按照来源权威性、版本和内容覆盖程度排序；高优先级资料不足时才降级补充。
- 答案使用证据约束的教学改写，不再机械复述原文，同时禁止补造手册没有支持的控制项、按键和步骤。
- 引用编号与手册页绑定，正文中可以嵌入对应原页，支持放大、滚轮翻页和打开原手册位置。
- 本地问答和主动联网搜索结果均写入持久缓存，相同问题优先直接返回。
- 普通问答固定使用 DeepSeek V4 Flash；用户主动选择联网搜索时使用 DeepSeek V4 Pro MAX。
- 已入库手册按 DCS 官方、Chuck 和用户资料分类，过滤战役、地图及无关文本。

## 手册库与首次使用

- 支持 PDF、DOCX、EPUB、HTML、Markdown、TXT 和 RTF。
- 首次处理提供阶段进度和当前文件提示，避免大量手册解析时被误认为程序无响应。
- 支持复制 DCS 客户端中的官方英文手册。
- 支持按机型选择或批量下载 Chuck's Guides。
- 增加重复文件检测，避免官方手册或用户手册被重复复制和索引。
- API Key、手册来源和模型策略集中到设置页管理。
- 增加清除用户设置和缓存后自动重启的入口。

## 桌面内置手册窗口

- 桌面浮窗与 VR 浮窗使用同一套问答、引用、原页和图片交互组件。
- 桌面模式恢复独立窗口尺寸，不再使用覆盖整个显示器的透明承载窗口。
- 标题栏支持原生窗口拖动。
- 输入框支持 Enter 提问和 Shift+Enter 换行。
- 关闭、再次呼出和 DCS 焦点恢复统一经过同一套状态入口。

## OpenXR VR 内置窗口

- 新增原生 OpenXR API Layer、共享内存帧桥和 D3D11 纹理提交。
- 自动跟随仪表板选择的桌面或 VR 启动模式。
- 每次呼出记录当前头部姿态，在 LOCAL 空间建立固定锚点；显示后不再持续跟随头部。
- 呼出时保留视线方向和俯仰，但去除 Roll 侧倾，使歪头呼出时画布仍保持水平。
- 标题栏拖动由平面平移升级为球面轨道运动：面板围绕呼出锚点移动、距离保持不变并持续朝向锚点。
- 水平轨道限制约为 ±65°，垂直轨道限制约为 ±35°，避免面板翻面或移动到用户身后。
- 提升帧捕获频率，并使用单帧背压防止低性能设备堆积截图任务。
- 修复关闭后迟到的异步帧重新激活窗口、快捷键关闭后无法再次呼出、隐藏窗口未归还 DCS 焦点等问题。
- OpenXR 共享协议升级到 V3，并增加原生姿态数学自检。

## 软件启动与设置

- 设置页增加统一的软件设置区域，集中管理预设、接入状态、路径、静默启动和启动延迟。
- 内置模块和用户软件均可选择普通或静默启动；未选择静默时直接按程序路径启动。
- 一键启动始终可用，只启动当前预设中尚未运行的软件，最后检查并启动 DCS。
- 每个软件可设置确认运行后的延迟，全部延迟结束后再启动游戏。
- 修复 MOZA、Pimax 等软件正常退出时的衍生服务判断和延迟关闭误报。
- 改进中文软件名称、图标提取、管理员权限和窗口最小化兼容性。

## 更新与发布

- 公开分发改为 Windows 安装版，绿色目录仅用于维护者本地调试。
- GitHub Actions 构建安装程序和 SHA-256 校验文件。
- 新增可由用户关闭的启动更新检查。
- 只有 Release 正文包含维护者推送标记时才弹出更新通知；普通提交不会主动打扰用户。
- 更新不会在后台自动下载安装，用户始终从官方 Release 页面确认下载。

## 模组管理器与界面

- 多个 DCS 游戏目录分别使用独立本地模组仓库，并提供全局统计。
- 新增全局模组预设、全部启用、全部停用、ZIP 导入和冲突处理。
- 支持备份 `Saved Games\DCS\Config` 并显示上次备份时间。
- 更新 DCSHUB 品牌图标、安装程序图标、侧边栏布局和中英文界面。
- 仓库主页改为正式的产品说明结构，并加入仪表板、超级手册、模组管理器和游戏内置窗口截图。

## 升级和兼容性说明

- 用户设置仍保存在 `%APPDATA%\dcs-control-hub`。
- 手册库、模组仓库和备份目录保持用户原有选择。
- DeepSeek API Key 继续使用 Windows 当前用户凭据加密保存。
- 本版本更新了 OpenXR API Layer。验证 VR 功能前必须彻底退出并重新启动 DCS，以确保游戏加载新版 DLL。
- 公开版本以 Windows 安装版为准；当前安装程序尚未进行商业代码签名。
